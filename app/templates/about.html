About the Project

The SLTC blends gesture recognition with an on-device conversational AI. It identifies hand gestures, converts them into readable text, and communicates with a chatbot that responds intelligently — all running locally for speed, privacy, and accessibility.

Vision

SLTC aims to bridge communication barriers by enabling real-time interaction between sign language users and digital systems. The system is fully open-source, lightweight, and optimized for edge devices.

Conversational Layer

The integrated chatbot runs locally using Ollama with Mistral as the LLM. This ensures fast, private responses without relying on cloud services, making conversations smooth and secure.

Core Technology Stack

SLTC combines gesture tracking and AI-driven conversation using the following technologies:

Flask TensorFlow MediaPipe OpenCV Ollama Mistral pyttsx3
